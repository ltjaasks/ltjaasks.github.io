Bug bounty-ohjelmien yhteydessä tutkin sivujen GraphQL schemoja. Ohjelmalla voi verrata vanhaa GraphQL schemaa uuteen pyydettyyn schemaan. Ohjelma vertaa schemoja toisiinsa ja selvittää tehdyt muutokset. Ohjelma ottaa parametrina vanhan scheman tiedoston ja uuden scheman tiedoston, vertaa tiedostoja ja tuottaa tekstitiedoston, joka sisältää tiedon lisätyistä, poistetuista ja muutetuista riveistä sekä rivinumerot, joilla muutokset ovat. Projekti githubissa
 When looking into bug bounty programs I investigated GraphQL schemas of the webapplications. This program allows comparing an old GraphQL schema to a new updated one. The program takes the old and new schemas file names as parameters, compares the files and produces a text file that contains information about added, deleted and modified lines and the line numbers where the modifications are.
